Real-Time Fraud Detection System

Distributed producer-consumer pipeline that streams simulated financial transactions through Redis Streams to two parallel consumers — a supervised XGBoost classifier and an unsupervised Isolation Forest — flagging fraud to MongoDB in real time. Includes a Faker-powered transaction generator producing 100,000+ PII-free, geo-accurate records for model training and load testing.

Stack: Python · Redis Streams · XGBoost · Isolation Forest · MongoDB · Faker · Docker

Model Drift Monitoring & Auto-Retraining Pipeline

Built an end-to-end MLOps pipeline that ingests production e-commerce shipping data via FastAPI, detects distribution drift using Kolmogorov-Smirnov statistical tests across 4 key features (p < 0.05 threshold), and enqueues a Celery retraining job automatically when drift is flagged and implemented Champion-Challenger promotion strategy with a custom business-cost metric (FN: $50, FP: $5) — challenger model trained on a sliding window of recent data replaces champion only when it achieves strictly lower validation cost; all runs logged to MLflow for full auditability

Stack: Python · OpenCV · Ultralytics YOLO · TensorFlow/Keras · NumPy · Docker · python-dotenv
